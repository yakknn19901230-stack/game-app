# スーパーフェイスラン 🏃💨

自分の顔写真がキャラクターになる、マリオ風の横スクロールアクションゲームです。
ブラウザだけで動きます(ビルド・サーバー不要)。

## 遊ぶ

📱 **iPhone・スマホ・PCのブラウザでここを開くだけ:**
https://yakknn19901230-stack.github.io/game-app/

iPhoneでは Safari の共有ボタン → 「ホーム画面に追加」をすると、
全画面のアプリとして遊べます。

> 🔒 プライバシーについて: 選んだ顔写真は自分の端末のブラウザ内にだけ保存されます。
> サーバーやGitHubにアップロードされることは一切ありません。

## ローカルで遊ぶ場合

1. `index.html` をブラウザで開く
2. タイトル画面で「顔写真をえらぶ」から好きな顔画像をアップロード
   (顔は自動で丸く切り抜かれてキャラクターの顔になります。選んだ顔はブラウザに保存され、次回も使えます)
3. 「ゲームスタート」を押してプレイ!

## 操作方法

| 操作 | キー |
|------|------|
| 移動 | ← → または A / D |
| ジャンプ | スペース / ↑ / W (長押しで大ジャンプ) |

スマホ・タブレットでは画面上のタッチボタンで操作できます。

## ルール

- **かんたん**(3さいから・おちる穴なし)と**ぼうけん**の2つのコースをタイトルで選べる
- 敵は上から踏むとたおせる。横からぶつかるとミス!
- ハテナブロックを下から叩くとコインやアイテム、レンガ(茶色)は壊せる
- アイテム: 🍄キノコ=大きくなって1回耐えられる / 🌸フラワー=ファイアボール(X/Zか🔥ボタン) / ⭐スター=8秒むてき
- 穴に落ちたり時間切れになるとミス。残機は3
- ゴールの旗にたどりつけばクリア! 残りタイムがボーナスになるよ

## 技術構成

- HTML5 Canvas + Vanilla JavaScript(依存ライブラリなし)
- 顔画像は中央を正方形に切り抜き256pxに縮小して `localStorage` に保存
- 効果音は WebAudio API で生成
